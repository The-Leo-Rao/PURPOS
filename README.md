# PURPOS

**PURPOS** is an Android application built to help NGOs streamline operations, publish requirements, manage volunteers, store organizational data, and generate analytics reports.

The app focuses on bridging the gap between NGOs and volunteers while also helping organizations maintain internal operational records in one place.

---

## Features

### NGO Authentication
- Secure signup and login using Firebase Authentication
- Password reset support
- Additional onboarding information collection during registration

### Volunteer & Requirement Management
- Publish NGO requirements/posts
- Match and view volunteers for specific needs
- Maintain success case records

### NGO Data Management
- Store internal NGO datasets
- Export operational data into CSV format
- Organize uploaded datasets in cloud storage

### Analytics Dashboard
- Select CSV files from storage
- Generate analytics reports
- External backend integration for advanced processing

### NGO Profile Management
- Edit organization details
- Upload profile image
- Manage location data using Google Places API
